# AI Debt Scanner Framework 🚀

This is a universal reasoning framework designed for AI agents (like Gemini, Claude, or GPT-based CLIs) to audit and clean technical debt generated by "vibe coding" in multi-language projects. Unlike traditional static analysis tools, this framework provides the agent with a **structured reasoning protocol** to detect patterns in code using its own intelligence and standard toolsets.

## Features

- 🔍 **Language Agnostic:** Works on any programming language by analyzing intent, structure, and semantic quality.
- 🧠 **Instructional Reasoning:** No external dependencies, binaries, or scripts required. The AI agent *becomes* the auditor.
- ⚖️ **Standardized Metrics:** Generates JSON reports with vulnerability mapping and complexity scores.
- 🛡️ **Universal Guardrails:** Acts as a quality gate whenever an AI writes or refactors code, regardless of the platform.

## How it Works

The framework implements a **Multi-Agent Reasoning Protocol**:
1.  **Pre-Writing Hook**: A mandatory validation step to ensure new code aligns with project conventions and modern standards (e.g., React 19, Python 3.12).
2.  **Scanner Mode**: The agent audits the codebase for AI artifacts, structural bloat, and lazy patterns.
3.  **Architect Mode**: The agent plans refactorings, ensuring atomic design and proper modularization.
4.  **Cleaner Mode**: The agent executes surgical changes and verifies the reduction in "Technical Debt Temperature".

## Usage

You can invoke this skill by asking the agent to:
- "Scan this project for AI debt."
- "Audit my code for vibe-coding patterns."
- "Clean up the technical debt in [file]."

The agent will output a standardized report and suggest a plan for improvement.

### Users can install it via:

    
    npx skills add sebamar88/ai-debt-scanner
