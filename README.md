# AI Debt Scanner Skill 🚀

This is an instructional skill for the Gemini CLI agent designed to audit and clean technical debt generated by AI ("vibe coding") in multi-language projects. Unlike traditional tools, this skill provides the agent with a **Reasoning Framework** to detect patterns in LLM-generated code using its own intelligence and standard CLI tools.

## Features

- 🔍 **Language Agnostic:** Works on any programming language by analyzing intent and structure.
- 🧠 **Instructional Reasoning:** No external dependencies or scripts required. The agent *is* the auditor.
- ⚖️ **Debt Scoring & Metrics:** Outputs standardized JSON with vulnerability mapping and complexity metrics.
- 🛡️ **Generation Guardrails:** Acts as a quality framework whenever the agent writes new code.

## How it Works

This skill defines a **Multi-Agent Protocol** within the AI's reasoning:
1.  **Pre-Writing Hook**: Automatically triggers before writing code to ensure alignment with project DNA and modern standards (e.g., React 19+).
2.  **Scanner Mode**: The agent audits the codebase, identifying hotspots like AI artifacts, structural bloat, and lazy patterns.
3.  **Architect Mode**: The agent plans refactorings, ensuring atomic design and proper abstraction.
4.  **Cleaner Mode**: The agent applies surgical refactorings and verifies that the "Debt Temperature" has decreased.

## Usage

You can invoke this skill by asking the agent to:
- "Scan this project for AI debt."
- "Audit my code for vibe-coding patterns."
- "Clean up the technical debt in [file]."

The agent will output a standardized report and suggest a plan for improvement.

### Users can install it via:

    
    npx skills add sebamar88/ai-debt-scanner
